# 孔头股票 - Kongtou Stock

一个功能完整的 A 股实时行情监控系统，支持板块分析、资金流向追踪、K线展示和智能告警。

## 🎯 主要功能

- 📊 **实时行情显示** - 实时获取 A 股个股行情数据
- 📈 **K线图表** - 展示股票的历史K线数据
- 🎨 **板块分析** - 查看各行业板块的涨跌情况
- 💹 **资金流向** - 追踪龙虎榜数据和北向资金流向
- 🔔 **价格告警** - 自定义告警规则，价格达到目标时提醒
- 📊 **数据查询** - 支持最长 2 年范围内的历史数据查询
- ⏰ **自动更新** - 每分钟自动更新行情数据

## 🛠️ 技术栈

### 后端
- **Python 3.10+**
- **FastAPI** - 异步 Web 框架
- **APScheduler** - 定时任务调度
- **AKShare** - A股数据源
- **SQLAlchemy** - ORM 数据库管理
- **SQLite** - 轻量级数据库

### 前端
- **Vue 3** - 现代前端框架
- **Vite** - 快速构建工具
- **Echarts** - 数据可视化
- **Axios** - HTTP 客户端
- **Element Plus** - UI 组件库

## 🚀 快速开始

### 后端安装

```bash
cd backend
pip install -r requirements.txt
python -m app.main
```

服务器将在 `http://localhost:8000` 启动，API 文档：`http://localhost:8000/docs`

### 前端安装

```bash
cd frontend
npm install
npm run dev
```

前端将在 `http://localhost:5173` 启动

## 📊 数据更新

系统会自动在每分钟的整点时刻更新以下数据（工作日 9:30-15:00）：
- ✅ 个股实时行情
- ✅ 板块指数
- ✅ 龙虎榜数据
- ✅ 北向资金流向
- ✅ 历史K线数据（每天）
